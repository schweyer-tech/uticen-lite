**Goal:** Deliver the 2026-06-27 functionality review as a sequential series of PRs — fix two server errors, polish the Logic Builder and data-source UX, move the control ID into the Definition tab, add node renaming + pop-out editors, rebrand the user-facing copy to "Uticen / Uticen Lite", and add a public-API demo control.

**Tech Stack:** Python ≥3.11, FastAPI, Jinja2 templates, HTMX, vanilla JS, CodeMirror (vendored), SQLite, pytest. ruff (py311, line-length 100) + mypy gates.

## Global Constraints

- **Cardinal contract (learning 0001):** No change may touch `contract/bundle.schema.json`, `bundle/`, `schema/`, or any bundle-facing field. Gates: `tests/test_contract_export.py` + `tests/schema/test_bundle_schema.py` stay green. Adding store state never bumps the **bundle** `schema_version`; if a store migration step is added, bump the **store** `SCHEMA_VERSION` in `store/migrations.py` in the same commit (learning 0022).
- **Dev gates (every task ends green):** `python -m pytest -q` (pristine, no new warnings), `python -m ruff check .`, `python -m mypy controlflow_sdk`. ruff target `py311`, line-length 100.
- **Rebrand is UI-copy-only.** Rename user-facing strings to **"Uticen"** (the product / full SaaS) and **"Uticen Lite"** (this local edition). Spelling is **Uticen** (not "Utiecen"). NEVER rename: the package `controlflow-sdk`, module `controlflow_sdk`, CLI `cflow`, command `controlplane`, the `"Generated by controlflow-sdk"` workpaper attribution, update/upgrade templates that name the package, or `pyproject.toml`.
- **Editors are server-rendered sub-route tabs** (learning 0007) — keep that idiom; no client-side tab frameworks.
- **Derived/in-progress graphs must degrade, never 500** (learning 0013).
- **Cosmetic node fields stay out of the content-addressed cache key** (learning 0030): a new node `title` is non-data like `narrative` — exclude it from the canonical key so renames don't bust the step cache.
- **Re-run the e2e browser smoke** (`pytest tests/e2e -m browser`) whenever an HTMX swap restructures a `plane/` form in place (learning 0012).
- **Untrusted authored code stays behind the three layers** (learning 0008) — no change to the Custom Python node may widen its capability.
