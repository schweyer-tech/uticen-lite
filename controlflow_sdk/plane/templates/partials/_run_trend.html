{#- Inline-SVG run trend (no JS, no chart dep). `trend.points` is chronological (oldest->newest, left-to-right). All colors route through var(--token) so the light theme works automatically (learning 0005). Layout: a fixed-coordinate viewBox sized by CSS (never `height="auto"`, which is an invalid SVG presentation attribute — B1). A gridded plot area carries the pass-rate line + per-run dots against a 0–100% Y scale; exception counts read as short bars hanging from the plot floor. A legend above names both series. -#} {%- set pts = trend.points -%} {%- set n = pts | length -%} {#- plot geometry (user units; the SVG scales to fit via viewBox) -#} {%- set PAD_L = 34 -%} {#- room for the Y-axis % labels -#} {%- set PAD_R = 10 -%} {%- set PLOT_TOP = 8 -%} {%- set PLOT_H = 70 -%} {#- pass-rate plot band (0%..100%) -#} {%- set BAR_BAND = 22 -%} {#- exception-bar band below the plot -#} {%- set AXIS_BAND = 14 -%} {#- date labels sit below the bars, never over them -#} {%- set W = 360 -%} {%- set PLOT_W = W - PAD_L - PAD_R -%} {%- set PLOT_BOT = PLOT_TOP + PLOT_H -%} {%- set H = PLOT_BOT + BAR_BAND + AXIS_BAND -%} {%- set max_failed = trend.max_failed -%} {#- map a 0..100 pass-rate to a Y coordinate inside the plot band -#} {%- macro yof(rate) -%}{{ (PLOT_TOP + (1 - rate / 100) * PLOT_H) | round(2) }}{%- endmacro -%} {%- macro xof(i) -%}{{ (PAD_L + (i / (n - 1) * PLOT_W if n > 1 else PLOT_W / 2)) | round(2) }}{%- endmacro -%}
